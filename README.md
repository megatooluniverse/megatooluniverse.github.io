# 🌌 MegaToolUniverse

**The Ultimate Internet Tools Operating System.** A premium, modern, and lightning-fast central hub to access 40+ internet tools from a single platform. Built with a stunning Glassmorphism UI and a seamless Single Page Application (SPA) experience.

🔗 **Live Demo:** [megatooluniverse.github.io](https://megatooluniverse.github.io/)

---

## ✨ Features

* 🚀 **Zero Backend & Ultra Fast:** Purely built with HTML, CSS, and Vanilla JavaScript. Hosted directly on GitHub Pages.
* 💾 **Smart State Management:** Uses browser `LocalStorage` to save user preferences without requiring a database or login.
    * 📌 **Pin Tools:** Pin favorite tools to the top of the homepage.
    * 🔖 **Save Tools:** Bookmark tools for quick access later.
    * 🕒 **Recently Used:** Automatically tracks and displays recently opened tools.
    * 🔥 **Trending Tools:** Dynamically highlights the most popular tools.
* 💻 **The "OS" Experience:** Tools open in a smooth, slide-up iframe panel without reloading the page. Direct links update via URL hashes (e.g., `/#tweetgenpro`).
* 🔍 **Command Palette & Live Search:** Press `Ctrl + K` to open a quick-access command palette, or use the live search bar with hidden smart keywords.
* 📱 **Perfectly Responsive:** Includes a custom mobile bottom dock, floating action buttons, and a dynamic grid system that adapts to any screen size.
* 🎨 **Premium UI/UX:** * Multi-color animated Aurora/Mesh gradient background.
    * Frosted glass (Glassmorphism) cards with smooth hover and 3D glow effects.
    * Custom scrollbars and modern typography.
    * 🌓 **Dark & Light Mode** toggle.
* 📲 **PWA Ready:** Installable as a standalone app on Mobile and Desktop.

   \`\`\`bash
   git clone https://github.com/megatooluniverse/megatooluniverse.github.io.git
   \`\`\`

3. Simply double-click `index.html` to open it in your browser, or use an extension like VS Code Live Server for the best experience.
